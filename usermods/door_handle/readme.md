The original model provided no handle for the door, so i made one. It is derivatived from the Voron 0.1 door handle. I also added fitting corner pieces.

BOM:
- 2\* M3\*10 screws
- 2\* M3 t-nuts or hammer nuts
- VHB tape
- 8\* 6 mm x 3 mm magnets
